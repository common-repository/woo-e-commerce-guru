== Description ==

Este plugin é um plugin de integração dados da sua plataforma de E-Commerce com a plataforma do E-CommerceGuru (https://ecommerceguru.com.br/). 
Com a instalação do plugin e criação de sua conta nós iremos exportar os dados de Clientes, Produtos, Vendas, Newsletter, Carrinhos Abandonados, 
dentro outros dados do seu E-Commerce para nossa plataforma. Nós não enviaremos nenhum dado da nossa plaaforma para seu E-Commerce, apenas enviamos
as informações citadas do seu E-Commerce para a nossa plataforma.

Você pode entender mais sobro os serviços oferecidos pela nossa plataforma no link: https://ecommerceguru.com.br/

Você pode também ter acesso a nossa política de privacidade e termos de uso aqui: 


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the EcommerceGuru screen to configure the plugin

== Criação de sua conta no E-CommerceGuru ==

Você pode criar sua conta diretamente pelo nosso plugin ou através do nosso site https://ecommerceguru.com.br/.

Caso já tenha uma conta criada ou decida criar a conta pelo nosso site, você deverá se conectar com nossa plataforma através do no plugin inserindo
login e senha na seção especificada ma tela de cofiguração para recuperar sua chave de integração, é com ela que realizaremos a integração entre as 
duas plataformas.
